这是一个卡色儿学院招生系统

View your app in AI Studio: https://ai.studio/apps/0129cc05-6f53-4f4c-80e1-80008bbc6319

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
